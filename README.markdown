# Pong with Pygame

A simple Pong game made with Pygame in Python.

## Features
- Ball accelerates until a maximum speed.
- Player controls:
  - Left player: `CTRL LEFT` (down), `SHIFT` (up)
  - Right player: Arrow keys (`↓` for down, `↑` for up)
- First to 10 win.

## Start
```
python main.py
```


Enjoy playing!
